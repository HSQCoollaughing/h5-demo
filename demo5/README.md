# 自适应页面

### 介绍

这个项目是一个利用媒体查询的demo，你可以在PC端下看到一个布局完美的页面，同时转换到移动端下，依然可以完美布局

主要部分的css代码

```css
@media screen and (min-width: 750px){
  //...
}

@media screen and (max-width: 750px){
  //...
}
```
